Kodi Update

 

 

Windows Vista, 7, 8 and 10

                                   

                                    Use Firefox browser to download and run kodi15.exe from this link http://is.gd/kodi15 to install the latest update for windows operating system.

                                   

                                    Follow the Instructions when you run Kodi.exe and it will install updates automatically.

 

Android

 

                             Step 1:  Download and install Official Kodi from Google play store, then run it once and exit the app.

                             Step 2:  Download and install ES File Explorer from Google play store.

                             Step 3:  Download zip file from http://is.gd/kodian in your internet browser.

                             Step 4:  Run ES File Explorer and extract Kodi.zip file to the location below on your android device.

 

                                         /sdcard0/Android/data/org.xbmc.kodi/files  (Tick “Apply to all” box and Overwrite when asked).

 

                   Amazon Fire TV/ Stick – Windows Setup Only

 

Amazon Account

 

If you don’t have an Amazon account then register on http://www.amazon.co.uk/  and also setup payment details to

install free apps from Amazon App Store.

                                                         

If you already have an Amazon account and haven’t setup payment details then you have to setup payment details to

install free apps from Amazon App Store.

 

Amazon Fire TV/Stick First Time Boot/Setup

 

Plug your Amazon Fire TV or Stick into the HDMI TV and Micro USB into external socket to power it up (If external power

doesn’t boot Fire TV Stick then plug the USB into the USB input on the side of TV to power up).

 

Follow the on screen setup to connect your WIFI and then login to your Amazon Account to register device.

 

Step 1:  Navigate from Home screen to Settings > Display & Sounds > Display >  Calibrate Display > SCALE TO YOUR SCREEN.

Step 2:  Go back to Settings > Applications > TURN OFF Collect App Usage Data.

Step 3:  Go back to Setting > System > Developer Options > TURN ON both ADB Debugging and Apps from Unknown Sources.

Step 4:  Go back to Settings > System > About > Network > WRITE OR MAKE NOTE OF THE IP ADDRESS E.G 192.168.?.???

Step 5:  Go back to Settings > Parental Controls > TURN ON AND SETUP PIN.

Step 6:  Also TURN ON Password-Protect Purchases and Password-Protect Video Playback TO AVOID ANY CHARGES.

Step 7: Go back to Home Screen and then proceed to your PC for the next steps.

 

Install Apps from Windows PC to Amazon Fire TV/Stick

                                     

On your Windows PC

 

Using Firefox Browser download and run KodiAFTV.exe from http://is.gd/kodiafs and follow the onscreen Instructions.

 

Step 1:  Press ENTER to download and extract required files on your PC (Program will do this automatically).

Step 2:  Type the IP Address which you noted down and press ENTER, it should say connected if unable to connect (Make sure both PC and Amazon Fire TV/Stick is on the same WIFI).

Step 3:  Press ENTER to install apps and copy files onto your Amazon Fire TV/Stick (Program will do this automatically).

Step 4: Leave program running and proceed to your Amazon Fire TV/Stick for the next steps.

 

On your Amazon Fire TV/Stick 

 

Step 1:  Navigate to Settings > Applications > Manage Applications > FireStarter > Launch Application TO START FireStarter App.

Step 2:  Go to FireStarter Settings on your left and at the bottom of settings click Import and then HOME BUTTON.

Step 3:  Run ES File Explorer App and inside the app navigate to Settings > Backup > CLICK ON RESTORE THEN SELECT OK AND EXIT ES File Explorer.

Step 4:  Run Kodi app for the first time and then exit the app straight away (this will create a folder needed for userdata).

Step 5:  Run ES File Explorer and select Amazondata.zip file at the bottom by holding down SELECT BUTTON on the remote, then press MENU BUTTON and select “Extract to” and custom then OK.

Step 6:  Select the extracted .kodi folder by holding down SELECT BUTTON and press MENU BUTTON and select “Move to” /Android/data/org.xbmc.kodi/files and select Yes to all and OVERWRITE ALL FILES and Exit after move is complete.

Step 7:  The Settings App is ONLY to connect Bluetooth devices e.g headphones, keyboard and mouse (not smartphones).

 

Follow the program on your PC to finish the setup and your Amazon Fire Stick will reboot to complete Install.
