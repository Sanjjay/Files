Kodi Update

 

 

Windows Vista, 7, 8 and 10

                                   

                                    Use Firefox browser to download and run kodi15.exe to install the latest update for windows operating system.

                                   

                                    Follow the Instructions when you run Kodi.exe and it will install updates automatically.

 

                        Android

 

                             Download and install Kodi from google play store, then run it once and exit the app.

 

                             Download and install ES File Explore from google play store.

                            

                             Download zip file from http://is.gd/kodian and extract to the location below on your android device.

                            

                             /sdcard0/Android/data/org.xbmc.kodi/files.
